# AgentForge

Gemma hackathon project under Maya AI in Paperclip.

## Mission
Turn existing OpenClaw, Hermes, and Paperclip-style agent workspaces into portable, shareable packages with Gemma 4 generated manifests.

## MVP
- Portable manifest schema
- Deterministic workspace scanner
- Gemma 4 manifest generation
- Preview metadata card
- Export package flow
- Import package flow

## Core demo
Scan a real agent workspace, generate a manifest, preview it, export it, and import it again with near-zero manual setup.
